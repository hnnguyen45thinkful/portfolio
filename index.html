Collabowrite

Real-time user interaction · React · ES2016 · Node.js · Express · MongoDB/Mongoose · Webpack

Collabowrite is an app that enables users to work together real-time on sentence/paragraph-level writing. It uses account-free sign-up/in by generating a random, unique project access code when a user first creates a project. The code can then be shared to let others join and contribute to the project by providing feedback, revisions, or new proposals, which get displayed real-time to all users in session. Combined with a mobile-friendly interface, Collabowrite is great for inviting participation from a large audience during presentations and class.

Lookitup

Contextualized word search · Open API use · AJAX calls

Lookitup is a word/phrase-search engine that goes above and beyond a dictionary definition and saves you the hassle of going through search results. Lookitup grabs the following:
1. Relevant dictionary entries along with pronunciation audio files and usage examples from Longman’s Dictionary;
2. The intro section of the term’s Wikipedia page when available;
3. YouTube videos filtered for informative clips, instantly playable on an on-page pop-up player.
